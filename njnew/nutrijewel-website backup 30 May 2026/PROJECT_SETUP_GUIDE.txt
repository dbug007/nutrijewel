TROUBLESHOOTING COMMON ISSUES
=============================

1. PORT ISSUES:
   - Ensure port 3366 is not in use
   - Use: lsof -ti:3366 to check
   - Kill process: kill -9 $(lsof -ti:3366)

2. DEPENDENCY ISSUES:
   - Clear npm cache: npm cache clean --force
   - Delete node_modules and package-lock.json
   - Reinstall: npm install

3. CSS NOT LOADING:
   - Check import statements in components
   - Verify CSS file paths are correct
   - Ensure CSS files exist for all components

4. WHATSAPP INTEGRATION:
   - Test on mobile devices for best experience
   - Verify phone number format: +91 996-063-7656
   - Check URL encoding for special characters
