### Image Specifications:
- **Size**: 800x600px minimum
- **Format**: JPG (for photos), PNG (for graphics with transparency)
- **Aspect Ratio**: 4:3 or 16:9
- **Quality**: High resolution, well-lit, appetizing

### Implementation Steps:
1. Download images from Unsplash/Pexels using suggested keywords
2. Rename files according to our naming convention
3. Place in `/public/images/` directory
4. Optimize file sizes (compress to ~200KB each)
5. Update image references in RecipesBlogPage.js
